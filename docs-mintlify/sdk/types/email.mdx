## Usage

Developers choose between two approaches:

<Tabs>
  <Tab title="Custom HTML">
    Provide custom HTML directly:

    ```typescript
    await stackServerApp.sendEmail({
      userIds: ["user-id-1"],
      subject: "Hello!",
      html: "<h1>Welcome</h1><p>Thanks for signing up!</p>",
    });
    ```

  </Tab>
  <Tab title="Template">
    Use a template with variable substitution:

    ```typescript
    await stackServerApp.sendEmail({
      userIds: ["user-id-1"],
      templateId: "welcome-template",
      variables: {
        userName: "John Doe",
        activationUrl: "https://app.com/activate/token123",
      },
    });
    ```

  </Tab>
</Tabs>

<Note>
  The `notificationCategoryName` and `themeId` properties work independently
  with either approach.
</Note>
