---
title: "Delete Current User"
description: "Deletes the currently authenticated user. Use this with caution."
api: "DELETE /users/me"
---

## Request

No request parameters or body required.

## Response

<ResponseField name="200" type="object">
  Successful response indicating the user was deleted.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.stack-auth.com/users/me" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
