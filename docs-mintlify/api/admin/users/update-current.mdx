---
title: "Update Current User"
description: "Updates the currently authenticated user. Only the values provided will be updated."
api: "PATCH /users/me"
---

## Request

The request body is required. Only the values provided will be updated.

## Response

<ResponseField name="200" type="object">
  Successful response containing the updated user information.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X PATCH "https://api.stack-auth.com/users/me" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"display_name": "Updated Name"}'
```
</CodeGroup>
