---
title: "Get Current User"
description: "Gets the currently authenticated user"
api: "GET /users/me"
---

## Request

No request parameters required.

## Response

<ResponseField name="200" type="object">
  Successful response containing the current user's information.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.stack-auth.com/users/me" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
