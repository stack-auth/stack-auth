---
title: "List Sessions"
description: "List all sessions for a user"
api: "GET /auth/sessions"
---

## Request

<ParamField query="user_id" type="string">
  The identifier for the user account. Use `"me"` to reference the currently authenticated user.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response containing session data for the specified user.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.stack-auth.com/auth/sessions?user_id=me" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
