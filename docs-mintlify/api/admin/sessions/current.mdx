---
title: "Sign Out Current Session"
description: "Sign out of the current session and invalidate the refresh token"
api: "DELETE /auth/sessions/current"
---

## Request

No request parameters or body required.

## Response

<ResponseField name="200" type="object">
  Successful response indicating the session has been terminated and the refresh token invalidated.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.stack-auth.com/auth/sessions/current" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
