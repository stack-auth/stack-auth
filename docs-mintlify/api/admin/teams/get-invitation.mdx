---
title: "Delete Team Invitation"
description: "Delete a team invitation by its identifier"
api: "DELETE /team-invitations/{id}"
---

## Request

<ParamField path="id" type="string" required>
  The invitation identifier to delete.
</ParamField>

<ParamField query="team_id" type="string">
  The team identifier. Required unless `user_id` is supplied.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response indicating the invitation was deleted.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X DELETE "https://api.stack-auth.com/team-invitations/INVITATION_ID?team_id=TEAM_ID" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
