---
title: "List Team Invitations"
description: "Retrieves a list of team invitations based on specified filters"
api: "GET /team-invitations"
---

## Request

<ParamField query="team_id" type="string">
  The identifier for the team whose invitations should be listed. Required unless `user_id` is specified.
</ParamField>

<ParamField query="user_id" type="string">
  Filters invitations sent to a user's verified email addresses. Must be set to `"me"` for client-level access. Cannot be used together with `team_id`.
</ParamField>

Either `team_id` or `user_id` must be provided, but not both simultaneously.

## Response

<ResponseField name="200" type="object">
  Successful response containing team invitation data based on the applied filters.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.stack-auth.com/team-invitations?team_id=TEAM_ID" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
