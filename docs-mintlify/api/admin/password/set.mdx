---
title: "Set Password"
description: "Set a new password for the current user."
api: "POST /api/v1/auth/password/set"
---

## Request

### Headers

<ParamField header="X-Stack-Project-Id" type="string">
  The unique identifier of the project.
</ParamField>

<ParamField header="X-Stack-Publishable-Client-Key" type="string">
  The publishable client key.
</ParamField>

<ParamField header="X-Stack-Access-Token" type="string">
  The access token for the authenticated user.
</ParamField>

### Body

<ParamField body="password" type="string" required>
  The new password to set for the user.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Always equal to true.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.stack-auth.com/api/v1/auth/password/set" \
  -H "Content-Type: application/json" \
  -H "X-Stack-Project-Id: <project-id>" \
  -H "X-Stack-Publishable-Client-Key: <client-key>" \
  -H "X-Stack-Access-Token: <access-token>" \
  -d '{
    "password": "newSecurePassword123"
  }'
```

```javascript JavaScript
const response = await fetch("https://api.stack-auth.com/api/v1/auth/password/set", {
  method: "POST",

    "Content-Type": "application/json",
    "X-Stack-Project-Id": "<project-id>",
    "X-Stack-Publishable-Client-Key": "<client-key>",

  },

    password: "newSecurePassword123"
  })
});
const data = await response.json();
```
