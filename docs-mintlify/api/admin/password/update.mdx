---
title: "Update Password"
description: "Update the password of the current user, requires the old password."
api: "POST /api/v1/auth/password/update"
---

## Request

### Headers

<ParamField header="X-Stack-Project-Id" type="string">
  The unique identifier of the project.
</ParamField>

<ParamField header="X-Stack-Publishable-Client-Key" type="string">
  The publishable client key.
</ParamField>

<ParamField header="x-stack-refresh-token" type="string">
  The refresh token for the authenticated user.
</ParamField>

### Body

<ParamField body="old_password" type="string" required>
  The current password of the user.
</ParamField>

<ParamField body="new_password" type="string" required>
  The new password to set.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Always equal to true.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.stack-auth.com/api/v1/auth/password/update" \
  -H "Content-Type: application/json" \
  -H "X-Stack-Project-Id: <project-id>" \
  -H "X-Stack-Publishable-Client-Key: <client-key>" \
  -H "x-stack-refresh-token: <refresh-token>" \
  -d '{
    "old_password": "currentPassword123",
    "new_password": "newSecurePassword456"
  }'
```

```javascript JavaScript
const response = await fetch("https://api.stack-auth.com/api/v1/auth/password/update", {

    "X-Stack-Publishable-Client-Key": "<client-key>",

  },

```
