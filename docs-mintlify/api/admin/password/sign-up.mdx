---
title: "Sign Up with Email and Password"
description: "Create a new account with email and password."
api: "POST /api/v1/auth/password/sign-up"
---

## Request

### Headers

<ParamField header="X-Stack-Project-Id" type="string">
  The unique identifier of the project.
</ParamField>

<ParamField header="X-Stack-Publishable-Client-Key" type="string">
  The publishable client key.
</ParamField>

### Body

<ParamField body="email" type="string" required>
  The email to sign up with.
</ParamField>

<ParamField body="password" type="string" required>
  The password for the new account.
</ParamField>

<ParamField body="verification_callback_url" type="string">
  The base callback URL to construct a verification link for the verification e-mail. A query parameter `code` with the verification code will be appended to it. The page should then make a request to the `/contact-channels/verify` endpoint.
</ParamField>

## Response

<ResponseField name="access_token" type="string" required>
  Short-lived access token that can be used to authenticate the user.
</ResponseField>

<ResponseField name="refresh_token" type="string" required>
  Long-lived refresh token that can be used to obtain a new access token.
</ResponseField>

<ResponseField name="user_id" type="string" required>
  The unique identifier of the newly created user.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.stack-auth.com/api/v1/auth/password/sign-up" \
  -H "Content-Type: application/json" \
  -H "X-Stack-Project-Id: <project-id>" \
  -H "X-Stack-Publishable-Client-Key: <client-key>" \
  -d '{
    "email": "johndoe@example.com",
    "password": "mySecurePassword123",
    "verification_callback_url": "https://example.com/handler/email-verification"
  }'
```

```javascript JavaScript
const response = await fetch("https://api.stack-auth.com/api/v1/auth/password/sign-up", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "X-Stack-Project-Id": "<project-id>",
    "X-Stack-Publishable-Client-Key": "<client-key>"
  },
  body: JSON.stringify({
    email: "johndoe@example.com",
    password: "mySecurePassword123",
    verification_callback_url: "https://example.com/handler/email-verification"
  })
});
const data = await response.json();
```
