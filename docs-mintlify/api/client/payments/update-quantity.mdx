---
title: "Update Item Quantity"
description: "Update the quantity of a specific item for a customer."
api: "POST /api/v1/payments/items/{customer_type}/{customer_id}/{item_id}/update-quantity"
---

## Request

### Path Parameters

<ParamField path="customer_type" type="string" required>
  The classification identifying the customer category.
</ParamField>

<ParamField path="customer_id" type="string" required>
  The unique identifier for the customer.
</ParamField>

<ParamField path="item_id" type="string" required>
  The unique identifier of the specific item to update.
</ParamField>

### Body

<ParamField body="body" type="object" required>
  The request body containing the updated quantity details.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response confirming the quantity update.
</ResponseField>
