---
title: "List Team Invitations"
description: "Retrieve a list of team invitations based on specified criteria."
api: "GET /api/v1/team-invitations"
---

## Request

### Query Parameters

<ParamField query="team_id" type="string">
  Identifier for the team whose invitations should be listed. Required unless `user_id` is provided. Cannot be used simultaneously with `user_id`.
</ParamField>

<ParamField query="user_id" type="string">
  Lists invitations directed to a specific user's verified email addresses. Must be set to `me` for client-level access. Cannot be used simultaneously with `team_id`.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response with the list of team invitations.
</ResponseField>
