---
title: "Update Team Member Profile"
description: "Update your own team member profile. user_id must be me in the path parameters on the client."
api: "PATCH /api/v1/team-member-profiles/{team_id}/{user_id}"
---

## Request

### Path Parameters

<ParamField path="team_id" type="string" required>
  The identifier for the team.
</ParamField>

<ParamField path="user_id" type="string" required>
  The user identifier, or `me` for the authenticated user.
</ParamField>

### Body

<ParamField body="body" type="object" required>
  The request body containing the profile fields to update.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response with the updated team member profile.
</ResponseField>
