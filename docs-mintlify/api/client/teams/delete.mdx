---
title: "Delete a Team"
description: "Delete a team. Only allowed if the current user is a member of the team and has the $delete_team permission."
api: "DELETE /api/v1/teams/{team_id}"
---

## Request

### Path Parameters

<ParamField path="team_id" type="string" required>
  The identifier of the team to be deleted.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response indicating the team has been deleted.
</ResponseField>
