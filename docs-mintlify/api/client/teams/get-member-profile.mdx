---
title: "Get Team Member Profile"
description: "Retrieve a team member's profile. Users can always access their own profile using me as the user_id. Viewing other profiles requires the $read_members permission."
api: "GET /api/v1/team-member-profiles/{team_id}/{user_id}"
---

## Request

### Path Parameters

<ParamField path="team_id" type="string" required>
  The identifier for the team.
</ParamField>

<ParamField path="user_id" type="string" required>
  The user's ID, or `me` to retrieve the current authenticated user's profile.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response with the team member profile data.
</ResponseField>
