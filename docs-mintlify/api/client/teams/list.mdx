---
title: "List Teams"
description: "Lists all the teams that the current user is a member of."
api: "GET /api/v1/teams"
---

## Request

### Query Parameters

<ParamField query="user_id" type="string" required>
  Filter for the teams that the user is a member of. Can be either `me` or an ID. Must be `me` in the client API.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response listing all teams associated with the authenticated user.
</ResponseField>
