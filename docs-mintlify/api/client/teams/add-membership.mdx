---
title: "Add Team Membership"
description: "Add a user as a member of a team."
api: "POST /api/v1/team-memberships/{team_id}/{user_id}"
---

## Request

### Path Parameters

<ParamField path="team_id" type="string" required>
  The identifier for the team.
</ParamField>

<ParamField path="user_id" type="string" required>
  The identifier for the user to add to the team.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response indicating the user has been added to the team.
</ResponseField>
