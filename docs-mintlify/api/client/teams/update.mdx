---
title: "Update a Team"
description: "Update a team. Only allowed if the current user is a member of the team and has the $update_team permission."
api: "PATCH /api/v1/teams/{team_id}"
---

## Request

### Path Parameters

<ParamField path="team_id" type="string" required>
  Identifier for the team to be updated.
</ParamField>

### Body

<ParamField body="body" type="object" required>
  The request body containing the fields to update.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response with the updated team details.
</ResponseField>
