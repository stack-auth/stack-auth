---
title: "List Project Permissions"
description: "Retrieves the current user's global permissions. For client-side requests, user_id must be set to me."
api: "GET /api/v1/permissions/project-permissions"
---

The combination of `user_id` and `permission_id` provides a unique identifier for each permission.

## Request

### Query Parameters

<ParamField query="user_id" type="string">
  Filter results by user ID. Client requests require `user_id=me`.
</ParamField>

<ParamField query="permission_id" type="string">
  Filter results by a specific permission ID.
</ParamField>

<ParamField query="recursive" type="boolean">
  Set to `true` to include direct and indirect permissions; set to `false` for direct permissions only.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response with the list of project permissions.
</ResponseField>
