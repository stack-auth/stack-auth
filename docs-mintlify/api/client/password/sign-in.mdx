---
title: "Sign In with Password"
description: "Sign in to an account with email and password."
api: "POST /api/v1/auth/password/sign-in"
---

## Request

<ParamField header="x-stack-publishable-client-key" type="string" required>
  The publishable client key for authentication.
</ParamField>

<ParamField header="x-stack-project-id" type="string" required>
  The project ID.
</ParamField>

<ParamField body="email" type="string" required>
  The email address of the account.
</ParamField>

<ParamField body="password" type="string" required>
  The password for the account.
</ParamField>

## Response

<ResponseField name="access_token" type="string">
  The access token for the authenticated user.
</ResponseField>

<ResponseField name="refresh_token" type="string">
  The refresh token for obtaining new access tokens.
</ResponseField>

<ResponseField name="user_id" type="string">
  The unique identifier of the authenticated user.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST 'https://api.stack-auth.com/api/v1/auth/password/sign-in' \
  -H 'Content-Type: application/json' \
  -H 'x-stack-publishable-client-key: YOUR_PUBLISHABLE_CLIENT_KEY' \
  -H 'x-stack-project-id: YOUR_PROJECT_ID' \
  -d '{
    "email": "user@example.com",
    "password": "mySecurePassword123"
  }'
```

```json 200 Response
{
  "access_token": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...",

}
```
</CodeGroup>
