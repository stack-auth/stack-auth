---
title: "Verify an Email"
description: "Verify an email address belonging to a user."
api: "POST /api/v1/contact-channels/verify"
---

## Request

<ParamField header="x-stack-publishable-client-key" type="string" required>
  The publishable client key for authentication.
</ParamField>

<ParamField header="x-stack-project-id" type="string" required>
  The project ID.
</ParamField>

<ParamField body="code" type="string" required>
  The verification code sent to the contact channel.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the verification was successful.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST 'https://api.stack-auth.com/api/v1/contact-channels/verify' \
  -H 'Content-Type: application/json' \
  -H 'x-stack-publishable-client-key: YOUR_PUBLISHABLE_CLIENT_KEY' \
  -H 'x-stack-project-id: YOUR_PROJECT_ID' \
  -d '{
    "code": "123456"
  }'
```

```json 200 Response
{
  "success": true
}
```
</CodeGroup>
