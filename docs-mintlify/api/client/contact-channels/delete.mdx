---
title: "Delete a Contact Channel"
description: "Removes a contact channel associated with a specified user."
api: "DELETE /api/v1/contact-channels/{user_id}/{contact_channel_id}"
---

## Request

<ParamField header="x-stack-publishable-client-key" type="string" required>
  The publishable client key for authentication.
</ParamField>

<ParamField header="x-stack-access-token" type="string" required>
  The access token for the authenticated user.
</ParamField>

<ParamField header="x-stack-project-id" type="string" required>
  The project ID.
</ParamField>

<ParamField path="user_id" type="string" required>
  The user that the contact channel belongs to, or `me` for the currently authenticated user.
</ParamField>

<ParamField path="contact_channel_id" type="string" required>
  The target contact channel to delete.
</ParamField>

## Response

Returns a `200` status code on success with an empty response body.

<CodeGroup>
```bash cURL
curl -X DELETE 'https://api.stack-auth.com/api/v1/contact-channels/me/cc_1234567890' \
  -H 'x-stack-publishable-client-key: YOUR_PUBLISHABLE_CLIENT_KEY' \
  -H 'x-stack-access-token: YOUR_ACCESS_TOKEN' \
  -H 'x-stack-project-id: YOUR_PROJECT_ID'
```

```json 200 Response
{}
```
</CodeGroup>
