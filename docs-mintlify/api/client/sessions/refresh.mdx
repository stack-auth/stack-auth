---
title: "Refresh Access Token"
description: "Get a new access token using a refresh token."
api: "POST /api/v1/auth/sessions/current/refresh"
---

## Request

### Headers

<ParamField header="x-stack-refresh-token" type="string" required>
  The refresh token used to request a new access token.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response containing the new access token.
</ResponseField>
