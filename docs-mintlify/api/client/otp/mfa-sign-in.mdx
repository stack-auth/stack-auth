---
title: "MFA Sign In"
description: "Complete multi-factor authorization to sign in, with a TOTP and an MFA attempt code."
api: "POST /api/v1/auth/mfa/sign-in"
---

## Request

<ParamField header="x-stack-publishable-client-key" type="string" required>
  The publishable client key for authentication.
</ParamField>

<ParamField header="x-stack-project-id" type="string" required>
  The project ID.
</ParamField>

<ParamField body="code" type="string" required>
  The TOTP code from the user's authenticator app.
</ParamField>

<ParamField body="attempt_code" type="string" required>
  The MFA attempt code received from the initial sign-in response.
</ParamField>

## Response

<ResponseField name="access_token" type="string">
  The access token for the authenticated user.
</ResponseField>

<ResponseField name="refresh_token" type="string">
  The refresh token for obtaining new access tokens.
</ResponseField>

<ResponseField name="user_id" type="string">
  The unique identifier of the authenticated user.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST 'https://api.stack-auth.com/api/v1/auth/mfa/sign-in' \
  -H 'Content-Type: application/json' \
  -H 'x-stack-publishable-client-key: YOUR_PUBLISHABLE_CLIENT_KEY' \
  -H 'x-stack-project-id: YOUR_PROJECT_ID' \
  -d '{
    "code": "123456",
    "attempt_code": "mfa_attempt_abc123..."
  }'
```
