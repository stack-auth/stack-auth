---
title: "Sign In with OTP Code"
description: "Sign in to an account using a one-time password code."
api: "POST /api/v1/auth/otp/sign-in"
---

## Request

<ParamField header="x-stack-publishable-client-key" type="string" required>
  The publishable client key for authentication.
</ParamField>

<ParamField header="x-stack-project-id" type="string" required>
  The project ID.
</ParamField>

<ParamField body="code" type="string" required>
  The one-time password code sent to the user's email.
</ParamField>

## Response

<ResponseField name="access_token" type="string">
  The access token for the authenticated user.
</ResponseField>

<ResponseField name="refresh_token" type="string">
  The refresh token for obtaining new access tokens.
</ResponseField>

<ResponseField name="user_id" type="string">
  The unique identifier of the authenticated user.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST 'https://api.stack-auth.com/api/v1/auth/otp/sign-in' \
  -H 'Content-Type: application/json' \
  -H 'x-stack-publishable-client-key: YOUR_PUBLISHABLE_CLIENT_KEY' \
  -H 'x-stack-project-id: YOUR_PROJECT_ID' \
  -d '{
    "code": "123456"
  }'
```

```json 200 Response
{

}
```
</CodeGroup>
