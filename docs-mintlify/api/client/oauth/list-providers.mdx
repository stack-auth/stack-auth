---
title: "List OAuth Providers"
description: "List all OAuth providers configured for the project."
api: "GET /api/v1/oauth-providers"
---

## Request

<ParamField header="x-stack-publishable-client-key" type="string" required>
  The publishable client key for authentication.
</ParamField>

<ParamField header="x-stack-project-id" type="string" required>
  The project ID.
</ParamField>

## Response

<ResponseField name="items" type="array">
  A list of OAuth providers.
  <Expandable>
    <ResponseField name="id" type="string">
      The unique identifier of the OAuth provider.
    </ResponseField>
    <ResponseField name="type" type="string">
      The type of the OAuth provider (e.g., `google`, `github`, `facebook`).
    </ResponseField>
    <ResponseField name="enabled" type="boolean">
      Whether the OAuth provider is currently enabled.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination information.
  <Expandable>
    <ResponseField name="next_cursor" type="string | null">
      Cursor for the next page of results.
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET 'https://api.stack-auth.com/api/v1/oauth-providers' \
  -H 'x-stack-publishable-client-key: YOUR_PUBLISHABLE_CLIENT_KEY' \
  -H 'x-stack-project-id: YOUR_PROJECT_ID'
```

</CodeGroup>
