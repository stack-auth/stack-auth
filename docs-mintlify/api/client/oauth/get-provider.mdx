---
title: "Get an OAuth Provider"
description: "Retrieves a specific OAuth provider by the user ID and the OAuth provider ID."
api: "GET /api/v1/oauth-providers/{user_id}/{provider_id}"
---

## Request

<ParamField header="x-stack-publishable-client-key" type="string" required>
  The publishable client key for authentication.
</ParamField>

<ParamField header="x-stack-access-token" type="string" required>
  The access token for the authenticated user.
</ParamField>

<ParamField header="x-stack-project-id" type="string" required>
  The project ID.
</ParamField>

<ParamField path="user_id" type="string" required>
  The ID of the user, or the special value `me` for the currently authenticated user.
</ParamField>

<ParamField path="provider_id" type="string" required>
  The OAuth provider identifier (e.g., `google`, `github`).
</ParamField>

## Response

<ResponseField name="provider_id" type="string">
  The OAuth provider identifier.
</ResponseField>

<ResponseField name="user_id" type="string">
  The ID of the user this OAuth connection belongs to.
</ResponseField>

<ResponseField name="account_id" type="string">
  The user's account ID on the OAuth provider.
</ResponseField>

<ResponseField name="email" type="string">
  The email address associated with the OAuth provider account.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET 'https://api.stack-auth.com/api/v1/oauth-providers/me/google' \
  -H 'x-stack-publishable-client-key: YOUR_PUBLISHABLE_CLIENT_KEY' \
  -H 'x-stack-access-token: YOUR_ACCESS_TOKEN' \
  -H 'x-stack-project-id: YOUR_PROJECT_ID'
```

```json 200 Response
{
  "provider_id": "google",
  "user_id": "usr_1234567890",
  "account_id": "1234567890",
  "email": "user@gmail.com"
}
```
</CodeGroup>
