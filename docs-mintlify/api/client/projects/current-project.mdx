---
title: "Get Current Project"
description: "Retrieve information about the current project, including display name, OAuth providers, and authentication methods."
api: "GET /api/v1/projects/current"
---

This endpoint is useful for displaying the available login options to the user.

## Request

No request parameters required.

## Response

<ResponseField name="200" type="object">
  Successful response with current project details including display name, OAuth provider configurations, and authentication method details.
</ResponseField>
