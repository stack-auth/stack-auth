## Event Type

`team.updated`

## Description

This event is triggered when a team is updated.

## Payload

<ResponseField name="type" type="string">
  The event type. Always `"team.updated"`.
</ResponseField>

<ResponseField name="data" type="object">
  The team object after the update.
  <Expandable>
    <ResponseField name="id" type="string">
      The unique identifier for the team.
    </ResponseField>
    <ResponseField name="display_name" type="string">
      The display name of the team.
    </ResponseField>
    <ResponseField name="created_at_millis" type="integer">
      The timestamp when the team was created, in milliseconds since epoch.
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```json Example Payload
{
  "type": "team.updated",
  "data": {
    "id": "team-id-123",
    "display_name": "Updated Team Name",
    "created_at_millis": 1700000000000
  }
}
```
</CodeGroup>
