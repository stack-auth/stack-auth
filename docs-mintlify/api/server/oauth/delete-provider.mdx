---
title: "Delete OAuth Provider"
description: "Remove an OAuth provider for a given user."
api: "DELETE /oauth-providers/{user_id}/{provider_id}"
---

## Description

Removes an OAuth provider for a given user.

## Path Parameters

<ParamField path="user_id" type="string" required>
  The ID of the user, or the special value `me` for the currently authenticated user.
</ParamField>

<ParamField path="provider_id" type="string" required>
  Identifier for the OAuth provider to remove.
</ParamField>

## Response

**Status Code:** `200` - Successful response
