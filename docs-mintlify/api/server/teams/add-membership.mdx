---
title: "Add User to Team"
description: "Add a user to a team"
api: "POST /team-memberships/{team_id}/{user_id}"
---

## Request

<ParamField path="team_id" type="string" required>
  Identifier for the team.
</ParamField>

<ParamField path="user_id" type="string" required>
  The user identifier, or `"me"` to reference the current authenticated user.
</ParamField>

The request body is required.

## Response

<ResponseField name="201" type="object">
  Successful response indicating the user was added to the team.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.stack-auth.com/team-memberships/TEAM_ID/USER_ID" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{}'
```
</CodeGroup>
