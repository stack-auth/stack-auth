---
title: "Get Team Member Profile"
description: "Get a team member profile by user ID"
api: "GET /team-member-profiles/{team_id}/{user_id}"
---

## Request

<ParamField path="team_id" type="string" required>
  The identifier for the team.
</ParamField>

<ParamField path="user_id" type="string" required>
  The ID of the user, or the special value `"me"` for the currently authenticated user.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response containing the team member's profile data.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.stack-auth.com/team-member-profiles/TEAM_ID/USER_ID" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
