---
title: "List Teams"
description: "List all the teams in the project"
api: "GET /teams"
---

## Request

<ParamField query="user_id" type="string">
  Filter for the teams that the user is a member of. Can be either `"me"` or a specific ID. Must use `"me"` in the client API.
</ParamField>

## Response

<ResponseField name="200" type="object">
  Successful response containing the list of teams in the project.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.stack-auth.com/teams?user_id=me" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</CodeGroup>
