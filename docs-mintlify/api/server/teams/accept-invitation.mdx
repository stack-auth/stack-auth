---
title: "Accept Team Invitation"
description: "Accept invitation and add user to the team"
api: "POST /team-invitations/accept"
---

## Request

The request body is required.

## Response

<ResponseField name="200" type="object">
  Successful response indicating the invitation was accepted and the user was added to the team.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.stack-auth.com/team-invitations/accept" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{}'
```
</CodeGroup>
