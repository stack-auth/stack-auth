api: "GET /emails/notification-preference/{user_id}"
---

## Description

Get all notification preferences for a user, showing which notification categories are enabled or disabled.

## Path Parameters

<ParamField path="user_id" type="string" required>
  The user identifier, or `me` to reference the currently authenticated user.
</ParamField>

## Response

**Status Code:** `200` - Successful response

Returns notification preference data for the specified user.
