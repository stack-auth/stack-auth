api: "GET /emails/outbox"
---

## Description

Retrieve a list of emails from the outbox, supporting optional filtering and pagination.

## Query Parameters

<ParamField query="status" type="string">
  Filter by email status.
</ParamField>

<ParamField query="simple_status" type="string">
  Filter by simplified status.
</ParamField>

<ParamField query="limit" type="integer">
  The maximum number of items to return. Maximum allowed is 100.
</ParamField>

<ParamField query="cursor" type="string">
  The cursor to start the result set from (email ID).
</ParamField>

## Response

**Status Code:** `200` - Successful response

Returns a paginated list of emails from the outbox.
