## Description

Deliver email messages to a list of recipients using one of three content delivery methods.

## Request Body

The `content` field accepts one of three formats:

### HTML Email

<CodeGroup>
```json HTML Content
{
  "content": {
    "html": "<html>...</html>"
  }
}
```

```json Template-Based Content
{
  "content": {
    "template_id": "string",
    "variables": {}
  }
}
```

```json Draft-Based Content
{
  "content": {
    "draft_id": "string"
  }
}
```
</CodeGroup>

## Response

**Status Code:** `200` - Successful response

Indicates the email request was processed successfully.
