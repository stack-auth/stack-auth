## Description

This endpoint enables a temporary increase to email sending capacity by a factor of 4x, remaining active for 4 hours.

## Response

<ResponseField name="success" type="boolean">
  Whether the capacity boost was successfully activated.
</ResponseField>

**Status Code:** `200` - Successful response
