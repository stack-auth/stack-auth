---
title: "Update Notification Preference"
description: "Enable or disable a specific notification category for a user."
api: "PATCH /emails/notification-preference/{user_id}/{notification_category_id}"
---

## Description

Enable or disable a specific notification category for a user.

## Path Parameters

<ParamField path="user_id" type="string" required>
  The ID of the user, or the special value `me` for the currently authenticated user.
</ParamField>

<ParamField path="notification_category_id" type="string" required>
  Identifier for the notification category to modify.
</ParamField>

## Request Body

A request body is required for this operation.

## Response

**Status Code:** `200` - Successful response
