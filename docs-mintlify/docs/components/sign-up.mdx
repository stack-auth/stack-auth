A component that renders a sign-up page with various customization options.

For more information, please refer to the [custom pages guide](../customization/custom-pages).

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `fullPage` | `boolean` | `false` | If true, renders the sign-up page in full-page mode. |
| `automaticRedirect` | `boolean` | `false` | If true, redirect to `afterSignIn`/`afterSignUp` URL when user is already signed in without showing the "You are signed in" message. |
| `noPasswordRepeat` | `boolean` | `false` | If true, removes the password confirmation field. |
| `extraInfo` | `React.ReactNode` | - | Additional information to display on the sign-up page. |
| `firstTab` | `'magic-link' \| 'password'` | - | Determines which tab is initially active. Defaults to 'magic-link' if not specified. |

## Example

```tsx
import { SignUp } from '@stackframe/stack';

export default function Page() {
  return (
    <div>
      <h1>Sign Up</h1>
      <SignUp
        fullPage={true}
        automaticRedirect={true}
        firstTab='password'
        extraInfo={<>By signing up, you agree to our <a href="/terms">Terms</a></>}
      />
    </div>
  );
}
```
