---
title: "<SelectedTeamSwitcher />"
description: Enables team selection functionality within applications, with optional URL navigation on team selection.
---

For a comprehensive guide on using this component, refer to our [Team Selection documentation](../concepts/team-selection).

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `urlMap` | `(team: Team) => string` | - | A function that maps a team to a URL. If provided, the component will navigate to this URL when a team is selected. |
| `selectedTeam` | `Team` | - | The initially selected team. |
| `noUpdateSelectedTeam` | `boolean` | `false` | If true, prevents updating the selected team in the user's settings when a new team is selected. |

## Example

```tsx
import { SelectedTeamSwitcher } from '@stackframe/stack';

export default function Page() {
  return (
    <div>
      <h1>Team Switcher</h1>
      <SelectedTeamSwitcher
        urlMap={(team) => `/team/${team.id}`}
        selectedTeam={currentTeam}
        noUpdateSelectedTeam={false}
      />
    </div>
  );
}
```
