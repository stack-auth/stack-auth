The `useStackApp` hook returns a `StackClientApp` object from the one that you provided in the [`StackProvider` component](/docs/components/stack-provider). If you want to learn more about the `StackClientApp` object, check out the [StackApp](/docs/sdk/objects/stack-app) documentation.

## Usage

```jsx
import { useStackApp } from "@stackframe/stack";

function MyComponent() {
  const stackApp = useStackApp();
  return <div>Sign In URL: {stackApp.urls.signIn}</div>;
}
```
