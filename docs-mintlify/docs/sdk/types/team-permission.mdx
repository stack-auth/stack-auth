The `TeamPermission` object represents a permission that a user has within a team. Currently, it contains only an `id` to specify the permission.

You can get `TeamPermission` objects by calling functions such as `user.getPermission(...)` or `user.listPermissions()`.

## Table of Contents

```typescript
type TeamPermission = {
  id: string;
};
```

## Properties

<ResponseField name="id" type="string">
  The identifier of the permission.
</ResponseField>

## Access

`TeamPermission` objects can be obtained through methods like `user.getPermission(...)` or `user.listPermissions()`.

See [User permissions methods](/docs/sdk/types/user#permissions) for full details on retrieving and checking permissions.
