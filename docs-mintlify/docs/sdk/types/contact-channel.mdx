`ContactChannel` represents a user's contact information, such as an email address or phone number. Some auth methods, like OTP/magic link or password, use contact channels for authentication.

On this page:

- [`ContactChannel`](#contactchannel)
- [`ServerContactChannel`](#servercontactchannel)

# `ContactChannel`

Basic information about a contact channel, as seen by a user themselves.

Usually obtained by calling [`user.listContactChannels()`](/docs/sdk/types/user#currentuserlistcontactchannels) or [`user.useContactChannels()`](/docs/sdk/types/user#currentuserusecontactchannels).

## Table of Contents

```typescript
type ContactChannel = {
  id: string;
  value: string;
  type: "email";
  isPrimary: boolean;
  isVerified: boolean;
  usedForAuth: boolean;

  sendVerificationEmail(): Promise<void>;
  update(options): Promise<void>;
  delete(): Promise<void>;
};
```

## Properties

<ResponseField name="id" type="string">
  The ID of the contact channel.
</ResponseField>

<ResponseField name="value" type="string">
  The value of the contact channel. If the type is `"email"`, this is an email
  address.
</ResponseField>

<ResponseField name="type" type='"email"'>
  The type of contact channel. Currently this is always `"email"`.
</ResponseField>

<ResponseField name="isPrimary" type="boolean">
  Whether the contact channel is the user's primary contact method. If an email
  is set to primary, it will be the value on `user.primaryEmail`.
</ResponseField>

<ResponseField name="isVerified" type="boolean">
  Whether the contact channel is verified.
</ResponseField>

<ResponseField name="usedForAuth" type="boolean">
  Whether the contact channel is used for authentication. If `true`, the user
  can use this contact channel with OTP or password to sign in.
</ResponseField>

# `ServerContactChannel`

Like `ContactChannel`, but includes additional methods and properties that require the `SECRET_SERVER_KEY`.

Usually obtained by calling [`serverUser.listContactChannels()`](/docs/sdk/types/user#serveruserlistcontactchannels) or [`serverUser.useContactChannels()`](/docs/sdk/types/user#serveruserusecontactchannels).

## Table of Contents

```typescript
type ServerContactChannel =
  // Inherits all properties from ContactChannel
  ContactChannel & {
    update(options): Promise<void>;
  };
```
