# `TeamProfile`

The `TeamProfile` object represents the profile of a user within the context of a team. It includes the user's profile information specific to the team and can be accessed through the `teamUser.teamProfile` property on a `TeamUser` object.

## Table of Contents

```typescript
type TeamProfile = {
  displayName: string | null;
  profileImageUrl: string | null;
};
```

## Properties

<ResponseField name="displayName" type="string | null">
  The display name of the user within the team context, or `null` if no display
  name is set.
</ResponseField>

<ResponseField name="profileImageUrl" type="string | null">
  The profile image URL of the user within the team context, or `null` if no
  profile image is set.
</ResponseField>

---

# `ServerTeamProfile`

The `ServerTeamProfile` object is currently the same as `TeamProfile`.

```typescript
type ServerTeamProfile =
  // Inherits all functionality from TeamProfile
  TeamProfile;
```

<Note>
  These are team-specific profile attributes, distinct from general user profile
  data. Users may have different display names and profile images within
  different teams.
</Note>
