# `CurrentUser`

Use `useUser()` to get `CurrentUser` (client). Use `stackServerApp.getUser()` to get `CurrentServerUser` (server).

# `ServerUser`

Like `CurrentUser`, but includes additional methods and properties that require the `SECRET_SERVER_KEY`.

# `CurrentServerUser`

Combines both `CurrentUser` and `ServerUser` - has both session authentication and server-side capabilities.
