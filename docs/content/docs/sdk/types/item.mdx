# `Item`

Represents a quantifiable resource that users or teams can own and consume.

Items can be retrieved through:
- [`user.getItem(itemId)`](./user.mdx#currentusergetitem) / [`user.useItem(itemId)`](./user.mdx#currentuseruseitem)
- [`team.getItem(itemId)`](./team.mdx#teamgetitem) / [`team.useItem(itemId)`](./team.mdx#teamuseitem)
