This is the SDK reference for Stack Auth's JavaScript SDK surface.

Use `@stackframe/stack` for Next.js, `@stackframe/react` for React Router, TanStack Start, and generic React apps, and `@stackframe/js` for Nuxt, SvelteKit, NestJS, Express, Hono, Cloudflare Workers, and other JavaScript runtimes. For setup instructions and framework recipes, see [Setup & Installation](../getting-started/setup.mdx). You can also use [our REST API](/api/overview) from any language.
