# Project Settings

The Settings section provides configuration options for your Stack Auth project, including authentication providers, security settings, and integrations.

## Overview

Customize your authentication experience by configuring:
- Authentication providers (OAuth, social login, etc.)
- Security policies and requirements
- Email templates and branding
- API keys and webhooks
- Project integrations

## Configuration Areas

### Authentication Providers
Set up social login and OAuth providers:
- Google, GitHub, Microsoft, etc.
- Custom OAuth providers
- Provider-specific settings
- Callback URL configuration

### Security Settings
Configure security policies:
- Password requirements
- Multi-factor authentication
- Session management
- Rate limiting

### Branding & Templates
Customize the user experience:
- Email templates
- Login page branding
- Custom domains
- Theme configuration

### API Configuration
Manage API access:
- API keys
- Webhook endpoints
- CORS settings
- Rate limiting

## Need Help?

Consult the Stack Companion for detailed configuration guidance and best practices.
